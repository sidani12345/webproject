Online Course

Era teknologi seperti ini membuat coding menjadi kebutuhan skillset penting untuk berkembang. Masalahnya, untuk kamu yang mau belajar coding namun tidak ada dasar IT sama sekali, kamu akan kebingungan untuk mulai dari mana dan minimnya penjelasan mengenai jargon atau istilah pemrograman.

Selesai dari kelas ini kamu akan lebih mudah untuk mengerti cara menggunakan berbagai bahasa pemrograman serta memiliki logika dasar membuat suatu program. Langkah dasar untuk menjadi programmer dimulai di kelas ini.
